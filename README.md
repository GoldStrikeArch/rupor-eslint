# Rupor ESLint Frontend

Правила Eslint для фронтед приложений

## Использование

### Устанока

```bash
npm i git+ssh://git@github.com/cyber-rom/rupor-eslint.git -D
```

### Использование
В файле .eslintrc.json добавьте в секцию extends строку "rupor" 
```bash
// .eslintrc.json

  "extends": [
    //....
    "rupor"
  ],
```
