# Rupor ESLint Frontend

Правила Eslint для фронтед приложений

## Использование

### Устанока

```bash
    npm i git+ssh://git@bitbucket.org:cyrm/rupor-eslint-frontend.git -D
```

### Использование
В файле .eslintrc.json добавьте в секцию extends строку "rupor" 
```bash
// .eslintrc.json

  "extends": [
    //....
    "rupor"
  ],
```
